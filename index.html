CSS

  • When a developer wants to style an element directly in the HTML file, they use inline CSS.
  • Internal style sheet is when an entire stylesheet is embedded directly in the HTML file.
  • The most common way to style is by using an external CSS style sheet, which allows developers to keep all their CSS rules in a separate file.
